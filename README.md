##  Project Overview
This project was a practical exploration of **Digital Literacy** through the role of a **Student Digital Ambassador**. Its main purpose was to help students understand how to use digital tools responsibly, build a professional online presence, communicate clearly, and stay safe in online spaces. The work was divided into five tasks, each focusing on a different skill area: creating an awareness infographic, setting up student profiles on professional platforms, practising coding on HackerRank, learning email etiquette, and studying UPI fraud prevention. All the tasks were brought together in a single GitHub repository to present the work in an organized and professional way. Overall, the project helped build both awareness and practical skills, while also reflecting on how digital habits can affect academic life, career growth, and online safety.
